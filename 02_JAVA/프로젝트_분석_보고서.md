# Java 학원 수업 자료 프로젝트 분석 보고서

## 📋 프로젝트 개요

이 프로젝트는 Java 프로그래밍 학습을 위한 체계적인 교육 자료로 구성되어 있습니다. 
기초 문법부터 고급 개념(JDBC, MVC 패턴)까지 단계적으로 학습할 수 있도록 구성되어 있습니다.

### 2. **02_JDBC** - 데이터베이스 연동

#### 2.1 JDBCProject - 기본 JDBC 프로젝트
- **구조**: MVC 패턴 적용
  - **model/**: 데이터 모델 (VO, DAO)
  - **view/**: 사용자 인터페이스 (Menu)
  - **controller/**: 비즈니스 로직 제어
  - **run/**: 실행 클래스
  - **test/**: JDBC 테스트 코드 (5개 파일)

#### 2.2 JDBCProjectWithService - Service 레이어 추가
- **구조**: MVC + Service 패턴
  - **service/**: 비즈니스 로직 처리
  - **common/**: 공통 유틸리티
  - **resources/**: 설정 파일
    - `driver.properties`: 데이터베이스 연결 정보
    - `query.xml`: SQL 쿼리 관리
    - `jdbcproject1.sql`: 데이터베이스 스키마
    - `JDBCReview.sql`: 복습용 SQL

#### 2.3 JDBCReview - 종합 리뷰 프로젝트
- **구조**: 완전한 MVC + Service 패턴
  - **model/**
    - `vo/`: Book, Member, Rental (Value Object)
    - `dao/`: BookDao, MemberDao, RentalDao (Data Access Object)
  - **service/**: BookService, MemberService, RentalService
  - **controller/**: BookController, MemberController, RentalController
  - **view/**: BookMenu (사용자 인터페이스)
  - **common/**: JDBCTemplate (공통 JDBC 유틸리티)
  - **resource/**
    - `book-query.xml`, `member-query.xml`, `rental-query.xml`: 분리된 쿼리 파일
    - `JDBCReview.sql`: 도서 대여 시스템 스키마
      - BOOK 테이블 (도서 정보)
      - MEMBER 테이블 (회원 정보)
      - RENTAL 테이블 (대여 정보, 외래키 관계)

## 🏗️ 아키텍처 패턴

### MVC 패턴 (Model-View-Controller)
프로젝트의 고급 예제와 JDBC 프로젝트에서 MVC 패턴을 적용하고 있습니다.

```
┌─────────────┐
│    View     │  ← 사용자 인터페이스 (Menu 클래스)
│  (화면 출력)  │
└──────┬──────┘
       │
       ↓
┌─────────────┐
│ Controller  │  ← 비즈니스 로직 제어
│  (흐름 제어)  │
└──────┬──────┘
       │
       ↓
┌─────────────┐
│    Model    │  ← 데이터 처리 (VO, DAO)
│  (데이터 관리) │
└─────────────┘
```

### Service 레이어 패턴
JDBCProjectWithService와 JDBCReview에서는 Service 레이어를 추가하여 
비즈니스 로직을 더욱 명확하게 분리했습니다.

```
View → Controller → Service → DAO → Database
```

2. 실습 문제를 먼저 풀어보고 정답 확인
3. example 패키지의 프로젝트를 분석하고 개선해보기
4. JDBC 프로젝트를 단계별로 따라하며 패턴 이해하기

배치관련 스타일

position : 상대위치(relative)와 절대위치(absolute)

첫번째 자식요소
두번째 자식요소
세번째 자식요소

position : relative

요소를 기본위치에 상대적으로 이동
이동 후에도 원리 자리공간은 그대로 차지함
주로 자식 absolute요소의 기준점을 만들기위해 사용

position : absolute

요소를 가장 가까운 position지정 조상요소(relative, absolute, fixed, sticky...)기준으로 위치
지정된 조상이 없다면 html 기준으로 위치
자신이 차지하던 공간을 무시하고 레이아웃에서 빠짐

position : 고정위치(fixed)

브라우저 뷰포트(화면)을 기준으로 위치가 고정됨.
스크롤시에도 항상 같은 위치값을 가짐.
레이아웃에서 제외(absolute와 동일하게 원래 공간 차지x)

float

요소를 왼쪽 또는 오른쪽으로 띄워서 배치
뒤에오는 컨텐츠는 띄워진 요소를 피해 흐름을 따라서 배치

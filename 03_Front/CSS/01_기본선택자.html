기본선택자

선택자란?

특정 html요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택해서 원하는 "스타일"과 "기능"을 적용시킬 수 있다.

1. 모든(전체) 선택자 : *

현재 이 문서(DOM Tree)상에 모든 요소들을 다 선택하고자 할 때

        *{
            스타일속성 : 값;
            스타일속성 : 값;
            ...
        }

2. 태그 선택자 : 태그명

현재 이 문서(DOM Tree)상에 해당하는 태그들을 전부 선택하고자 할 때 

        태그명{
            스타일속성 : 값;
            스타일속성 : 값;
        }

3.아이디 선택자 : #아이디

id 속성 : 해당 문서(DOM)에 딱 하나만 가지고있는 고유 식별 속성
문서상에서 특정 html요소 하나만 선택하고자 할 때 사용

  1. 리스트 출력
  2. 리스트 출력
  3. 리스트 출력
  4. 리스트 출력

4. 클래스 선택자 : .클래스명

class속성 : 특정 요소들을 묶어서 가져오기 위해 부여하는 식별자(그룹식별명방식)
문서상에서 내가 원하는 요소 "여러개"를 선택하고자 할 때 사용

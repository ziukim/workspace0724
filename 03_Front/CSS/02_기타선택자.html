1. 속성 선택자

선택하고자하는 요소에 작성되어있는 속성을 이용해서 선택하는 방법
선택자뒤에 []를 이용해서 속성과 속성값을 제시하여 선택 

        선택자[속성=속성값]
        선택자[속성~=속성값]
        선택자[속성|=속성값]
        선택자[속성^=속성값]
        선택자[속성$=속성값]
        선택자[속성*=속성값]{
            스타일속성 : 속성값;
            ...
        }

2. 자식선택자와 후손선택자

요소들이 중첩되어 작성된 경우
자식 : 바로 하위에 있는 요소 / 후손 : 하위요소 전부 

        자식을 선택 : >
        a>b{
            스타일속성 : 값;
            ...
        }
        후손을 선택 : 공백
        a b{
            스타일속성 : 값;
            ...
        }

    div태그의 자식이면서 후손

  • ul태그의 자식이면서 div태그의 후손

3. 동위(같은레벨) 선택자

동위관계에 위치한 특정 요소를 선택할 때 사용

        a요소 바로뒤있는 b요소 하나만 선택
        a+b{
            스타일속성:값;
            ...
        }

        a요소 뒤에 모든 b요소 선택
        a~b{
            스타일속성:값;
            ...
        }

4. 반응선택자

사용자의 움직임에 따라 선택되는 선택자

        해당 요소에 클릭이 되었을 경우 스타일을 부여하
        선택자:active{
            스타일속성:값;
            ...
        }

        해당 요소에 마우스가 올라가는 순간 스타일 부여
        선택자:hover{
            스타일속성:값;
            ...
        }

5. 상태 선택자

요소의 상태에 따라서 선택되는 선택자

        1) 체크된 상태의 요소
        선택자:checked{
            스타일속성:값;
            ...
        }

        2)초점이 맞춰진 input요소
        선택자:focus{
            스타일속성:값;
            ...
        }

        3)활성, 비활성 상태를 선택
        선택자:enalbed{
            스타일속성:값;
            ...
        }

        선택자:disalbed{
            스타일속성:값;
            ...
        }

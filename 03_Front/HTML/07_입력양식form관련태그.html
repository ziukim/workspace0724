폼(form) 관련 태그

form태그는 사용자로부터 입력을 받아 서버에 전달하는 영역을 정의하는 HTML태그
form태그내에submit버튼을 클릭시 해당 form안에 작성된 input요소의 값들을 전부 서버로 전달하는 요청을 보냄
form action="" method=""

- action : 해당 form에 사용자가 입력한 값들을 전달하면서 요청을 보낼 서버의 경로
- method : 요청 전송방식을 지정하는 속성(get(기본값), post)
> get : 요청시 사용자가 입력한 값들을 head에 담아 전송한다. -> url에 값이 전부 노출
> post : 요청시 사용자가 입력한 값들을 HTTP 본문(body)에 숨겨서 전송 -> url에 값이 노출x

form 내의 submit버튼을 클릭 시 action에 지정된 서버로 사용자가 입력한 값들이 전달되면서 요청됨.

text와 관련된 input태그의 타입

type="text"

- 한 줄 짜리 텍스트를 입력할 수 있는 텍스트상자
- maxlength(최대 길이), placeholder(입력 창 설명), required(필수 입력)등의 속성과 함께 사용.

type="password"

- 입력 한 문자를 ***형태로 가려줌, 비밀번호 입력에 사용, 속성은 text와 동일하게 사용

이외 텍스트관련 타입

여러 줄 입력 : textarea

- 여러 줄의 텍스트를 한 번에 입력 가능
- cols, rows로 크기 지정, resize:none으로 크기조절 제한 가능

숫자와 관련된 input태그의 타입

type="number"

- 숫자만 입력 가능, 스핀박스 표시
- min, max, step등의 속성 사용 가능.

type="range"

- 슬라이드바를 통해 숫자를 입력.

날짜 및 시간 입력

type = "date | month | week | time | datetime-local"

Select Box

드롭다운 목록에서 무조건 하나를 선택

radio button

- 여러 항목 중 하나만 선택 가능.
- 같은 name속성값을 가진 라디오 버튼끼리 그룹화

- 동시에 여러값을 선택할 수 있음
- 같은 name을 부여하면 배열형태로 서버에 전달됨.
